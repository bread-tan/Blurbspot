Blurbspot

Welcome to the project "Blurbspot!"

About

A blurb is a short summary or promotional piece accompanying a creative work.

Blurbspot! is a website, which gives you the latest Blurbs about your favourite bands. This project aims to simplify updating a user on their favourite artist or band. This is done by displaying to the user, the biography, images, social media interactions and songs of the artist. By doing so, the user is kept up to date with all the recent happenings of their favourite band.

Architecture

  • Front End: Made in PHP and stylized using Bootstrap 3.0.
  • Back End: Consists of python scripts to crawl, parse and scrape the web for static as well as dynamic data from various sources.
